LEDGERGO - WINDOWS LAUNCHER VERSION

This package is Windows-friendly, but it is NOT a true .exe build.
I added launcher BAT files so you can run it like an app on Windows.

HOW TO RUN
1. Install Node.js once on your PC.
2. Extract this ZIP.
3. Open folder: LEDGERGO Offline PC
4. Double-click:
   Start LEDGERGO.bat

It will:
- start the local server
- open browser automatically at http://localhost:3000

TO STOP
Double-click:
   Stop LEDGERGO.bat

NOTE
If you want a true .exe later, this project can be packed with:
- Electron + electron-builder
or
- pkg / nexe
on a Windows machine.
