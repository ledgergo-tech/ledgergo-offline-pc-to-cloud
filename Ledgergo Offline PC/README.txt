LEDGERGO - OFFLINE PC VERSION

Yeh version bina internet ke chalne ke liye banaya gaya hai.
Isme koi npm package dependency nahi hai.
Sirf Node.js install hona chahiye.

-----------------------------------
INSTALL / RUN
-----------------------------------
1. Node.js install karo (ek baar).
2. ZIP extract karo.
3. Folder open karo.
4. Terminal / CMD me:
   cd LEDGERGO-Offline-PC
   node server.js

5. Browser me open karo:
   http://localhost:3000

-----------------------------------
FEATURES
-----------------------------------
- Dashboard with Monthly Sales & Recent Transactions
- Customers Management (Balance Tracking & Statements)
- Items Management (Stock, Category with auto-suggestions, Low Stock Alerts)
- Bank Accounts Management
- Sales Invoices with Tax, Discounts, and Stock updates
- Purchase Entry & History (Stock auto-increase)
- Expense Tracking (Category-wise)
- Tax Invoice & Voucher Printing (Sales, Purchase, Expense)
- Reports: Sales Report, Daybook, Profit & Loss
- Local JSON file storage (Privacy & Speed)
- Truly Offline Mode (No internet required)

-----------------------------------
DATA STORAGE
-----------------------------------
Data local files me save hota hai:
- data/customers.json
- data/products.json
- data/banks.json
- data/invoices.json

Internet ki zaroorat nahi.

-----------------------------------
NOTES
-----------------------------------
- Yeh offline local version hai.
